# Gestão de Condomínio

sistema web para gerenciamento de consulta e reclamacoes de moradores em condominio

## 📌 Objetivo
Centralizar e organizar o registro de reclamações e sugestões de moradores em condomínio.

## ⚙️ Funcionalidades
- Cadastro de usuários  
- Login de usuários  
- Registro de reclamações  
- Acompanhamento de status  
-historico de solicitacoes 

## 🛠️ Tecnologias
- Frontend: HTML, CSS, JavaScript  
- Backend: Python  
- Banco de Dados: SQL ou MySQL  

## 📂 Estrutura do Projeto
- backend: lógica do sistema  
- frontend: interface do usuário  
- database: estrutura do banco de dados  
- docs: documentação
